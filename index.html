Claude Code Web Wrapper へようこそ！ 🚀

このデモページでは、設計した画面レイアウトを確認できます。

主な機能

  • モード切り替え: Chat, Code, Edit, Tool の4つのモード
  • 3行入力欄: マルチライン入力に対応
  • Y/Nボタン: 確認が必要な場合に表示
  • レスポンシブデザイン: モバイル・タブレット・デスクトップ対応

使い方

  1. 下部の入力欄にメッセージを入力
  2. モードボタンで機能を切り替え
  3. Enterキーまたは送信ボタンで送信
  4. 必要に応じてY/Nボタンで応答
